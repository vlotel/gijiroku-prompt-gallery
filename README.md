# 議事録プロンプトギャラリー

議事録や打ち合わせの文字起こしを、生成AIで見やすい1枚画像にまとめるためのプロンプト集です。
カードをクリックするとそのスタイルのプロンプトがコピーされ、末尾の貼り付け欄に会議内容を貼って画像生成AI（ChatGPT / Gemini など）に渡すだけで、見やすい資料画像が作れます。

整理済みの議事録でも、ボイスレコーダー等の文字起こし・メモでも利用できます。

## 使い方

1. `index.html` をブラウザで開く
2. 好きなデザインのカードをクリック（プロンプトがコピーされます）
3. コピーしたプロンプトの末尾に、議事録や文字起こしを貼り付け
4. 生成AIの画像生成に貼り付けて実行

## 収録スタイル（全17種）

「手書き・アナログ」「ビジネス・きっちり」「ビジュアル・華やか」「シンプル・上品」の4カテゴリ、計17スタイルを収録しています。
各カードには実際の生成見本画像を表示しています。

## 構成

| ファイル / フォルダ | 内容 |
|---|---|
| `index.html` / `style.css` / `app.js` | ギャラリー本体（静的Webアプリ） |
| `prompts.js` | 各スタイルのプロンプト（`プロンプトデータ/*.txt` を元に生成） |
| `プロンプトデータ/` | プロンプトの正本（.txt） |
| `見本画像/` | 各スタイルの生成見本画像 |

## プロンプト設計のポイント

- 固有名詞・数値・日付などの事実は改変・創作しない（正確な転記を最優先）
- 未整理の文字起こしは要点を分類・整理してよい（フィラーや雑談は省略）
- 未確定・要確認の項目は内容ベースで自動省略
- 日本の新字体で表記（簡体字の混入を防止）、見切れ防止
